# 关系模型（Relational Model）

`关系模型`（Relational Model）：基于`谓词逻辑`和`集合论`的一种数据模型。

`关系模型`的基本假定是所有数据都表示为数学上的`关系`，就是说 n 个集合的笛卡儿积的一个子集；有关这种数据的推理通过二值（就是说没有 NULL ）的`谓词逻辑`来进行，这意味着对每个命题都有两种可能的賦值：要么是真要么是假。

数据通过`关系演算`和`关系代数`方式进行操作。

> 关系演算（谓词）
>
> * 元组关系演算（行）
> * 域关系演算（列）
>
>   关系代数（集合）
