# 网络（Network）

## 通信机制

* RPC（Remote Procedure Call）：Protocol Buffer、Thrift、Avro（IDL、Serialization）
* Message：Queue、Pub-Sub（Pull、Push）
* Multi Broadcast：Gossip（Best Effort、Anti-Entropy<Push、Pull、Push-Pull>、Rumor Mongering）
